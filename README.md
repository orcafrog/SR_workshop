## 📚 개요
+ 주가 데이터를 활용하여 주가의 변동 패턴을 학습하고 주가 가격과 방향성을 예측하기<br>
적합한 ***LSTM(Long)*** 모형을 이용하여  18년치 데이터를 test set과 train set으로 분리하고<br>
1000회 학습시킨 후 이익 주가의 향방을 예측하였습니다. <br>
